# Augur 

Augur is a Python application for automatic zebra finch song detection. Using a lightweight (<250K parameter) CNN, it can quickly and reliably distinguish zebra finch song from other common noises encountered when recording zebra finches in the lab, no pretraining required.

## Features

- Can automatically search folders of recordings for files containing song
- Can be used to detect and save zebra finch singing in real time
- Features a GUI for easy usage
